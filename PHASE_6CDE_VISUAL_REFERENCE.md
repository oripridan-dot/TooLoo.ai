## Endpoint Quick Reference

```bash
# VELOCITY TRACKING
curl http://127.0.0.1:3012/api/v1/analytics/velocity-enhanced?domain=consensus
curl http://127.0.0.1:3012/api/v1/analytics/predict?domain=consensus&target=85
curl http://127.0.0.1:3012/api/v1/analytics/acceleration?domain=consensus

# BADGES
curl http://127.0.0.1:3012/api/v1/analytics/badges-full?userId=user1
curl http://127.0.0.1:3012/api/v1/analytics/badge-suggestions?userId=user1

# LEADERBOARDS
curl http://127.0.0.1:3012/api/v1/analytics/leaderboard/consensus?limit=20
curl http://127.0.0.1:3012/api/v1/analytics/leaderboard-overall
curl http://127.0.0.1:3012/api/v1/analytics/peer-comparison?userId=user1

# UPDATE PROFILE
curl -X POST http://127.0.0.1:3012/api/v1/analytics/learner-profile \
  -d '{"userId":"u1","username":"alex","masteryByDomain":{"consensus":92},...}'
```
