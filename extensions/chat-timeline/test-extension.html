🧪 Testing Instructions

Step 1: Load Extension
  1. Open Chrome and go to chrome://extensions/
  2. Enable "Developer mode" (top right toggle)
  3. Click "Load unpacked"
  4. Select the chrome-extension folder
  5. Verify extension appears with 📍 icon
Step 2: Test on ChatGPT
  1. Go to chat.openai.com
  2. Start a conversation (5+ message exchanges)
  3. Look for "📍 Timeline" button near input area
  4. Click button to open timeline sidebar
  5. Verify conversation segments appear
  6. Test "Jump to start" buttons
Step 3: Test Premium Flow
  1. Click "⭐ Upgrade" button in timeline
  2. Verify upgrade modal appears
  3. Check feature comparison table
