## üñ•Ô∏è The Workstation

The **Workstation** is the new unified interface for TooLoo.ai, designed to keep you in flow. It replaces the scattered tools with a single, 4-panel layout.

### Access
Navigate to: `http://127.0.0.1:3000/workstation`

### Layout
1.  **Task Panel (Top Left)**:
    *   **Current Objective**: Define what you are working on.
    *   **Checklist**: Break it down into steps.
    *   **Progress**: Track completion.
2.  **Chat Panel (Bottom Left)**:
    *   **Unified Chat**: Talk to TooLoo (orchestrated across all providers).
    *   **Multi-Format**: Choose between Markdown, JSON, or CSV responses.
    *   **Reactions**: Quickly reply or react to messages.
3.  **Context Panel (Top Right)**:
    *   **System Status**: See which services are active.
    *   **Active Context**: View loaded files and memory.
4.  **Artifacts Panel (Bottom Right)**:
    *   **Visualizations**: See Mermaid diagrams, concept maps, and flashcards generated by the AI.
    *   **File Preview**: View generated code or documents.

### Key Features
*   **Concept Maps**: Ask TooLoo to "draw a concept map of X" to see a visual representation in the Artifacts panel.
*   **Flashcards**: Ask for "flashcards about X" to get a study aid.
*   **Response Formats**: Use the dropdown in the chat input to force structured output (e.g., for data extraction).

## üîå Plugins

TooLoo.ai now supports a **Plugin System** to extend capabilities without modifying the core.

*   **Repo Manager**: Automatically organizes your branches and PRs.
*   **Custom Integrations**: Add your own tools via the `plugins/` directory.

### Managing Plugins
Plugins are managed via the **Capabilities Server**.
Check status: `http://127.0.0.1:3009/api/v1/plugins`
